Notations
==========

.. list-table::
    :widths: 30 70
    :header-rows: 1

    * - Notation
      - Description
    * - :math:`\textit x`
      - A scalar
    * - :math:`\mathbf{\textit x}`
      - A vector
    * - :math:`\mathbf{\textit X}`
      - A matrix
    * - :math:`\mathbf{1}`
      - A vector of all 1's
    * - :math:`\mathbb{R}`
      - A set of real numbers
    * - :math:`\mathcal{N}`
      - Normal distribution
